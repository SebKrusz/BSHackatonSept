BandSite - Sprint 3

Project Overview


There is now a working API that you can use to connect your front-end code to the back-end and get live data from the server. Your task is to now make the site functional with comments and shows data from the server, where before you had hard-coded mock data.


To submit this deliverable, create a new branch called sprint-3 where you will complete your assignment, then ensure you merge your sprint-3 branch with the main branch. Submit the link to your GitHub repository in Synapse before the submission deadline.

Tools & Resources


As before, be sure to use Sass to compile your project’s CSS. If you don’t have a Sass compiler installed, please see the installation instructions in the Content tab in Synapse.


There are no new design mockups, as there are no UI changes to the front-end.

Back-End API


Postman is a web client that makes testing APIs simple. You can use it to test the provided back-end API before writing any code.


When communicating with the API, you will need to supply an API key in order to authenticate yourself. Instructions for how to work with the API including registering to retrieve an API key can be found at the API documentation link above.
